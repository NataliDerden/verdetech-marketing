# Verde Email Campaign

Этот скилл создаёт полный пакет email-кампании для VerdeTech под автоматизацию в Битрикс24.

## Когда использовать

- Запуск рассылки на новый сегмент (например: "полуфабрикатные заводы ПФО")
- Создание email-кампании под конкретный оффер или продукт
- Улучшение существующей email-последовательности
- Подготовка материалов под конкретного крупного лида

## Входные данные

Спроси у пользователя (если не указано):

1. **Сегмент или лид** — на кого пишем? (тип производства, регион, конкретный завод)
2. **Оффер** — что предлагаем? (образцы, конкретный продукт, замена импорта)
3. **Что нужно на выходе** — полный пакет или отдельная часть:
   - `всё` — полный пакет (по умолчанию)
   - `исследование` — только профиль сегмента
   - `письма` — только email-последовательность
   - `продажи` — только материалы для менеджеров

## Порядок работы

### Шаг 1: Исследование сегмента

Запусти агента `verde-lead-researcher` с задачей:

> Исследуй сегмент [название сегмента]. Прочитай VERDE-KNOWLEDGE-BASE.md и PRD.md. Построй профиль: JTBD, триггеры, боли, возражения, конкурентный контекст, VOC-цитаты. Выдай рекомендации для email-кампании.

Результат: профиль сегмента с рекомендациями.

### Шаг 2: Email-последовательность

Запусти агента `verde-email-writer` с задачей:

> На основе профиля сегмента [передай результат шага 1] напиши email-кампанию для Битрикс24:
> - 1 холодное письмо (первое касание)
> - 3 follow-up письма (дни 3-5, 7-10, 14-15)
> - 1 nurture-письмо (после скачивания лид-магнита)
> - Таблицу последовательности с триггерами для Битрикс24
>
> Оффер: [оффер]. Сегмент: [сегмент].

Результат: готовые тексты писем + параметры автоматизации.

### Шаг 3: Материалы для менеджеров

Запусти агента `verde-sales-support` с задачей:

> На основе профиля сегмента [передай результат шага 1] создай:
> - Скрипт звонка на день 2 (после холодного письма)
> - Карточку обработки возражений (топ-5 для этого сегмента)
> - Одностраничник для этого сегмента
> - Шаблон КП с конкретными позициями
>
> Сегмент: [сегмент]. Оффер: [оффер].

Результат: пакет материалов для менеджера.

```markdown
## 3. Материалы для менеджеров
[результат verde-sales-support]

---

## Чек-лист запуска в Битрикс24
- [ ] Создать сегмент "[название]" в CRM
- [ ] Загрузить контакты в сегмент
- [ ] Настроить email-шаблоны (письма 1-4)
- [ ] Настроить триггеры и условия остановки
- [ ] Создать автоматическую задачу менеджеру на день 2 (звонок)
- [ ] Прикрепить скрипт звонка и карточку возражений к задаче
- [ ] Тестовая отправка на внутренний адрес
- [ ] Запуск
```

## Сохранение результатов

Сохрани итоговый файл в папку проекта:
`email-campaigns/[сегмент]-[дата].md`

Если папки `email-campaigns/` нет — создай.
