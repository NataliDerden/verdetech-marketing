## Изученные конкуренты

1. **ЭКОС** (ecosgp.com) — 700+ продуктов, 600+ клиентов, с 2005
2. **Ван Хеес Рус / SPEKO** (vhrus.ru) — немецкие рецептуры, только мясо, ISO + Халяль
3. **ГК Союзоптторг** (soyuzopttorg.com) — дистрибьютор, 3000+ позиций, 14 офисов
4. **ЭДВАНТА** (advanta-cs.ru) — комплексный поставщик (добавки + упаковка + санитария)
5. **Планета Вкуса** (plvk.ru) — прямой конкурент в СПб, схожий ассортимент
6. **Profi.Bio** (profi.bio) — биотехнологические добавки для мясопереработки

## Главный вывод

VerdeTech выигрывает на **скорости отгрузки** ("5 рабочих дней"), **гибкости объёмов** и **импортозамещении** — ни один конкурент не заявляет эти три УТП одновременно так явно. Это и есть основа для всех email-кампаний.
