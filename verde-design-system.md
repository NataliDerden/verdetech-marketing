# Verde Tech — Design System (машиночитаемый брендбук)

> Источник: `VERDE/VERDE гайдлайн PDF/Verde - гайдлайн.pdf` (A.LAB и D.A. SEVER, 2024)
> Плюс наблюдения со страниц визуализаций (биллборд, соцсети, транспорт, упаковка)
> Цель: чтобы ИИ-дизайнер (Midjourney / Flux / Figma / Gamma / Claude) мог генерировать материалы в фирменном стиле без человека на каждом шаге.

---

## 1. ДНК бренда

- **Название:** VERDE®TECH (вордмарк всегда капс, с символом ®)
- **Слоган:** «Забота о будущем» / «Caring for the Future»
- **Концепция:** «свежий взгляд на производство натуральных ингредиентов»
- **Тональность:** технологично, минималистично, уверенно, без пафоса и украшательств
- **Позиционирование:** альтернативные и технологичные решения в пищевой сфере — супер-продукты (безопасные + полезные + экономически выгодные партнёрам)
- **Продуктовые ДНК-сообщения:**
  - Современные технологии в пищевой промышленности
  - Натуральные термостабильные красители
  - Пищевые ингредиенты с дополнительной ценностью для здоровья
  - Инновации: витамин D, Mg, Инулин
  - Собственный цех для производства красителей

## 2. Логотип

### Файлы (где взять)
- Все версии: `VERDE/VERDE логотипы/` (Чёрные, Зелёные, Белые)
- На русском: `VERDE/Вердэ лого на русском/` (AI, SVG, PNG — включая в кривых)
- Аватары: `VERDE/VERDE аватары для соцсетей/` (Цветной/Монохром белый/чёрный — PNG + SVG)

### Компоновки
1. **Только знак** (4 лепестка) — для аватаров, фавиконов, угловых марок
2. **Только вордмарк** VERDE®TECH — для ситуаций, где знак невозможен
3. **Знак + вордмарк** — основная версия
4. **Знак + вордмарк + дескриптор в строку** (CARING FOR THE FUTURE / ЗАБОТА О БУДУЩЕМ)
5. **Знак + вордмарк + дескриптор в 2 строки** — для вертикальных форматов

### Охранное поле
- A = 1/2 B, где B — полная высота логоблока
- Применяется ко всем версиям, включая с дескриптором
- **Правило для ИИ:** вокруг логотипа всегда «воздух» ≥ половины его высоты

### Минимальные размеры (digital)
- Знак: **20px**
- Вордмарк: **80px**
- Основной логоблок: **120px**
- С дескриптором в строку: **200px**
- С дескриптором в 2 строки: **180px**

### Запрещено
- Менять пропорции, наклонять, искажать
- Использовать на низкоконтрастном фоне (см. §4 «Комбинации с фоном»)
- Добавлять тени, обводки, эффекты
- Применять градиенты к знаку

## 3. Палитра

| Роль | HEX | CMYK | Применение |
|---|---|---|---|
| Основной светлый | `#FFFFFF` | 0/0/0/0 | Фон, текст на тёмном |
| Основной тёмный | `#000000` | 0/0/0/100 | Текст, логотип, заливка |
| **Акцентный** | `#0FBF00` | 73/0/100/0 | Активные состояния, кнопки, заливки для выделения |
| Дополнительный | `#FCFCFC` | 0/0/0/1 | Софт-фон (чуть тёплее белого) |

### Комбинации с фоном (из гайдлайна, стр. 15)
- Белый фон → чёрный лого ИЛИ зелёный лого
- Чёрный фон → белый лого ИЛИ зелёный лого
- Зелёный фон → белый лого ИЛИ чёрный лого
- Серый (#FCFCFC) фон → чёрный лого ИЛИ зелёный лого
- **На других фонах** → только монохром (чёрный/белый), соблюдая контраст

### Наблюдение из визуализаций
В биллборде и соцсетях используется **жёлто-горчичный акцент** (≈ #F2B42A) как фоновая плашка под продуктовое фото. Это не входит в официальную палитру, но встречается. **Правило:** использовать как «продуктовый» фотофон, не применять к UI или лого.

## 4. Типографика

### Гарнитура
- **Onest** — единственный шрифт бренда
- Файлы: `VERDE/VERDE шрифты/Onest/` (variable + static + OFL лицензия)
- Поддерживает кириллицу и латиницу
- Лицензия: SIL Open Font License (бесплатная коммерция)

### Начертания (из гайдлайна)
- Black — заголовок/акциденция
- Bold — заголовок/подзаголовок
- Medium — подзаголовок
- Regular — основной текст
- Thin — акцидентные крупные надписи

### Рекомендации (экстраполяция на практику)
- **H1 / биллборд:** Onest Bold или Black, межстрочный 1.1
- **H2:** Onest Bold
- **H3 / подзаголовок:** Onest Medium
- **Body:** Onest Regular, интерлиньяж 1.4–1.5
- **Caption / подпись:** Onest Regular, размер 70% от body
- **Вордмарк логотипа**: капс, без украшений (зашит в SVG)

### Анти-правила
- Не использовать курсив (в гайдлайне не предусмотрен)
- Не подчёркивать, кроме ссылок
- Не смешивать с другими шрифтами

## 5. Композиция и фотостиль

### Наблюдения из визуализаций
- **Биллборд (стр.28):** разделение полотна вертикально 50/50 — слева белое поле с чёрным заголовком + логоблок со слоганом внизу, справа крупный продукт (капуста) на жёлтом фоне
- **Соцсети (стр.21-22):** квадрат, продукт на цветном однотонном фоне (жёлтый, зелёный), снизу белый вордмарк + слоган + знак в правом нижнем углу
- **Транспорт:** белый или зелёный фургон, вордмарк + слоган сбоку, знак отдельно
- **Визитки:** белый картон, знак + вордмарк слева, короткий слоган справа в 3 строки капсом, без графических украшений
- **Упаковка:** зеркальная металлизированная подложка + белая этикетка с №, названием, знаком, слоганом

### Фотостиль
- **Продукты:** крупный план, один объект, естественные цвета, минимальная тень
- **Фоны для фото:** плоские однотонные, желательно «тёплые» ноты (жёлтый, тёплый зелёный, серый)
- **Не использовать:** градиенты, сложные текстуры, фотомонтажи, стоки с людьми в костюмах
- **Люди (спикеры, сотрудники):** поясной портрет, спокойный фон, белый халат для технологов, поло/футболка с лого для выставок

### Сетки и модули
- Основная сетка: 12 колонок для digital, поле ≥1/2 высоты модуля
- Принцип «воздуха»: минимум 30–40% белого пространства на любом макете
- Горизонтальные линии-разделители — характерный приём гайдлайна (видно на каждой странице)

## 7. Tone of Voice (коммуникация)

### Как пишем
- Короткие уверенные фразы
- Факт + польза + технология (например: «Витамин D. Суточная норма в 50 мг премикса. Срок хранения 24 месяца.»)
- Избегаем «мы гордимся», «лидеры рынка», «инновационная компания»
- B2B-язык: технолог ≠ потребитель. Пишем для главного технолога мясокомбината

### Слоганы
- **Рус:** «Забота о будущем» (основной), «Свежий взгляд на производство натуральных ингредиентов» (дескриптор)
- **Eng:** «Caring for the Future»

### Что НЕ делаем
- Health claims без сносок (это уже отработано в Академии — все заявления с источниками)
- Абсолютные конкурентные утверждения («лучшие», «единственные»)
- Эмодзи в корпоративных материалах

## 8. Правила для ИИ-генерации (крит-важно)

### Что ИИ **может** делать сам
- Генерить фоны, фотореалистичные продукты, иллюстрации в монохроме
- Комбинировать элементы по сетке
- Генерить продуктовые карточки, слайды, посты
- Делать raster-варианты (PNG/JPG) на основе SVG-лого

### Что ИИ **не должен** делать
- Перерисовывать логотип — только вставлять готовый из `VERDE/VERDE логотипы/`
- Писать текст на вордмарке VERDE®TECH — всегда брать готовый SVG
- Подбирать шрифты — только Onest из `VERDE/VERDE шрифты/Onest/`
- Использовать цвета вне палитры #FFFFFF / #000000 / #0FBF00 / #FCFCFC (исключение — жёлто-горчичный продуктовый фон)
- Добавлять текстуры, градиенты, свечения, blur на основной макет

### Проверочный чек-лист (бренд-гардиан)
Перед публикацией любого макета ИИ/человек проверяет:
1. [ ] Логотип взят из исходников, не перерисован
2. [ ] Охранное поле логотипа соблюдено (A=1/2B)
3. [ ] Палитра: только #FFF/#000/#0FBF00/#FCFCFC (+ жёлтый для фото)
4. [ ] Шрифт Onest везде, без примесей
5. [ ] Контраст лого и фона соответствует §3
6. [ ] Слоган без искажений: «Забота о будущем» / «Caring for the Future»
7. [ ] Минимум 30% белого пространства
8. [ ] Нет теней, градиентов, украшательств
9. [ ] Для печати: CMYK, 300dpi, вылеты 3-5мм
10. [ ] Для digital: sRGB, оптимизация веса
