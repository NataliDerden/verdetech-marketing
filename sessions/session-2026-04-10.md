### 2. Демонстрация скиллов

**Cold Email** — написали холодное письмо для технолога мясозавода:
- Тема: "стабильность поставок ингредиентов"
- 4 абзаца, ~80 слов
- Принципы: пишем как коллега, лидируем миром клиента, один мягкий CTA
- Психология: social proof, zero-price effect, status-quo bias

**Lead Magnets** — спроектировали лид-магнит:
- Формат: чек-лист PDF, 2 страницы
- Тема: «Как заменить импортный ингредиент без переделки рецептуры»
- 5 шагов с конкретными действиями
- Гейтинг: email + имя + тип производства
- Дистрибуция: P.S. в cold email, звонок менеджера, сайт
- Связка с воронкой: чек-лист → email-sequence → образцы → сделка

### 4. Карта триггеров Битрикс24

Создали `bitrix24-trigger-map.md` — полная связка стадий воронки с автоматическими действиями.

**Воронка ЛИДОВ (по реальным стадиям клиента):**

| Стадия | Триггеры |
|---|---|
| 1. Не обработан | L1: автоответ (5 мин), L2: пинг менеджеру (2 ч) |
| 2.1. Взял в работу | Только задача менеджеру |
| 2.2. Не удалось связаться | L3: email сразу, L4: email 3 дня (кейс), L5: эскалация 7 дней |
| 3. Сделка создана | L6: email конвертации |
| Отказ лида | Без письма |

**Воронка СДЕЛОК:**

| Стадия | Триггеры |
|---|---|
| 1. Связь установлена | D1 (только без лида), D2 (по чекбоксу образцов), D15 (зависла 14 дн.) |
| 2. Образцы отработаны | D3: обратная связь |
| 3. КП согласовано | D4: подтверждение |
| 4. Счёт выставлен | D5: счёт, D6: напоминание 3 дня, D7: эскалация 7 дней |
| 5. Заказ произведён | D8: уведомление |
| 6. Оплата получена | D9: подтверждение |
| 7. Сделка закрыта | D10: NPS (через 1 день) |
| 8-11. Отказы | D11-D14: по причине отказа |

**Итого: 21 автоматическое действие** (15 писем + 4 уведомления + 1 задача + 1 NPS)

**Решения:**
- D1 vs L6: при конвертации лида → только L6, D1 → только для сделок без лида
- Стадия "Взял в работу": без автоматического письма, L1 достаточно

### 5. Аудит и переписывание шаблонов писем

Проверили все 21 шаблон из `bitrix24-email-templates.md` по принципам скиллов.

**Проблемы найденные в оригинальных шаблонах:**
- Письма начинались с "О нас:" вместо мира клиента
- Фичи вместо выгод ("R&D с ИТМО" вместо "подберём под вашу рецептуру")
- Отсутствие психологии убеждения (social proof, loss aversion, reciprocity)
- Шаблонные фразы ("Мы по-прежнему готовы помочь")
- Пассивный тон ("Хотели уточнить")
- Дублирование между письмами

**Переписано 10 шаблонов полностью:**

| Шаблон | Главное изменение | Скиллы |
|---|---|---|
| L1 | "О нас" → "Что вы получите" + social proof (300+ производств) | copywriting + marketing-psychology |
| L3 | Список контактов → оффер образцов первым абзацем | copywriting + cold-email |
| L4 | Повтор L3 → новый угол: кейс МитСтаб20 | email-sequence + cold-email |
| L6 | "Рады сотрудничеству" → "Вот что вы получите" + CTA | copywriting |
| D1 | Убран дубль "о нас", короткое и без повторов | copywriting |
| D3 | "Хотели уточнить" → "Пока свежий — зафиксируем" + сроки | marketing-psychology (urgency) |
| D6 | "Напоминаем" → "Заказ ждёт запуска" + 5 дней до отгрузки | marketing-psychology (loss aversion) |
| D9 | 2 строки → конкретный план: производство → уведомление → отгрузка | email-sequence |
| D10 | "Сделка завершена" → "Заказ доставлен" + крючок повторного заказа | marketing-psychology (peak-end rule) |
| D11 | "Понимаем, цена важна" → reframing: "В цену входит..." | marketing-psychology (reframing + social proof) |

**Мелкие правки в 6 шаблонах:**
- D2: + social proof
- D4: + commitment & consistency + конкретный срок
- D5: + specificity (5 дней до отгрузки)
- D8: + крючок на повторный заказ
- D12: убрана навязчивость
- D14: + конкретный срок (3 месяца)
