Главные правила

  • Только Onest во всех макетах
  • Палитра: только #000 / #FFF / #0FBF00 / #FCFCFC
  • Логотип — только из исходников, не перерисовывать
  • Охранное поле A=½B вокруг логотипа
  • Минимум 30% белого пространства
  • Слоган: «Забота о будущем» / «Caring for the Future»
  • Без теней, градиентов, украшательств
  • Для печати: CMYK, 300dpi, вылеты 3-5мм
