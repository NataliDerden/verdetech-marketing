# Бриф: Рыбопереработка

## Типичные под-линейки клиента
- Рыбные пресервы (селёдка, скумбрия, килька в масле/томате)
- Копчёная рыба (холодное и горячее копчение)
- Рыбные полуфабрикаты для HoReCa (филе, стейки, нарезка)
- Рыбные консервы (тушка в масле, натуральная)

## Что предлагать — по линейкам

### Рыбные пресервы
- **Основной:** Маринад Универсал + Маринад Французский
- **Bundle:** Натуральные красители (экстракт красной свёклы, морковь), V Smart Plus Mg (магний + граната)
- **Почему:** Универсал — нейтральный для экспортных, Французский — премиум (прованские травы). Натуральные красители = Clean Label. Граната — природный консервант.

### Копчёная рыба
- **Основной:** Маринад Грузинский (натуральный дым + травы) или Авторский (NDA)
- **Bundle:** Натуральные красители (карамель, бета-каротин), V Smart Plus Mg (граната как антиоксидант), экстракты натуральные
- **Почему:** Натуральный дым = тренд премиум. Магний + граната = функциональность + природный антиоксидант (вместо BHA/BHT).

### Рыбные полуфабрикаты (HoReCa)
- **Основной:** Маринад Универсал + моноспеции (если шеф использует)
- **Bundle:** Натуральные красители, V Smart Plus Витамины (базовый)
- **Почему:** HoReCa требует быстро и стандартно. Универсал на любой рыбе. Моноспеции = повар комбинирует сам.

### Рыбные консервы
- **Основной:** Маринад Универсал + моноспеции (корица, перец, лавр)
- **Bundle:** V Smart Plus Витамины, натуральные красители
- **Почему:** Консервы = замкнутая система, нужна стабильность маринада на длинном сроке. Витамины — для премиум-линеек (Омега-3 + витамин D).

## Типичные боли 2026
- Импортные маринады ЕС +25-30%, логистика через Беларусь
- Роспотребнадзор ограничил синтетические ароматизаторы, натуральный дым экологично
- X5/Магнит требуют убрать E-добавки
- Срок годности пресервов зависит от антиоксидантов, натуральные дорогие
- Специфика рыбы: запах, окисление жира, потемнение

## Возражения и ответы
- **«Дорого»** → «Замена импортного маринада экономит 15% себестоимости маринада. На 100 т продукции = +500-800 тыс ₽»
- **«Нужны особые компоненты»** → «Авторский маринад за 5 рабочих дней. Моноспеции для тестирования бесплатно»
- **«Clean Label, но антиоксиданты дорогие»** → «Экстракт розмарина стоит как синтетические E-добавки, но лучше для имиджа. Срок пресервов +2 мес»
- **«Ждут импортозамещения»** → «Образцы за 2 дня. 3-4 недели тест → если нравится, первая серия. Параллельно с импортом»

## Конкуренты
- **ЭКОС** — в т.ч. для рыбы. Слабость: дорого, долго
- **Ван Хеес Рус (SPEKO)** — рыбные маринады (200+). Слабость: 7-10 дней, старая немецкая школа
- **Местные производители** — дёшево, но нестабильное качество

## Тренды 2026-2028
- **2026:** Натуральный дым вытесняет синтетические ароматизаторы. Clean Label маринады = требование ритейла
- **2027:** Функциональные рыбные (омега-3, вит. D, магний) входят в премиум. Антиоксиданты из трав (розмарин, виноград) = норма
- **2028:** Рыбные консервы с комбо-функциональностью = мейнстрим в премиуме для 40+

## Реальные клиенты (кейсы)
- **Производитель рыбных пресервов (Северо-Запад)** — замена импортных маринадов, -15% к цене
- **Федеральный бренд копчёной рыбы** — рассол с антиоксидантами, +2 мес срок хранения
- **ООО из Мурманска** — универсал-маринад для HoReCa

## Benchmark ориентиры
- Цена маринада для рыбы: 100-150 ₽/кг (импорт), Verde 70-100 ₽/кг
- Срок поставки конкурентов 10-14 дней vs Verde 5 дней
- Срок годности пресервов: 6-12 мес → с натуральными антиоксидантами +2-3 мес
- Доля Clean Label маринадов: 12% (2024) → 40% (2026) → 70% (2028)
