# Бриф: Птицепереработка

## Типичные под-линейки клиента
- Филе куриное (замороженное / охлаждённое)
- Крылья и ножки (из импорта замена)
- Котлеты и наггетсы (ready-to-fry)
- Копчёная куриная грудка (премиум)

## Что предлагать — по линейкам

### Филе куриное
- **Основной:** Рассол для птицы (V Инъекта Супер)
- **Bundle:** Маринад Универсал, V Smart Plus D, Маринад Чесночный
- **Почему:** Грудка теряет 25-30% при термообработке без рассола → с рассолом 12-15%. Критично для замороженных полуфабрикатов.

### Крылья и ножки
- **Основной:** Маринад Универсал + Маринад Французский (комбо)
- **Bundle:** Рассол для птицы, V Smart Plus Витамины (базовый), натуральные экстракты
- **Почему:** Эти части сухие — маринад + рассол = сочность. Прованские травы = ресторанное качество в ритейле.

### Котлеты и наггетсы
- **Основной:** Рассол бесфосфатный (V Инъекта Супер без сои)
- **Bundle:** V Smart Plus Витамины с пребиотиками, V Smart Plus D с клетчаткой
- **Почему:** Котлеты требуют Clean Label. Пребиотики + клетчатка улучшают текстуру, снижают расслоение при жарке.

### Копчёная куриная грудка
- **Основной:** Авторский маринад (NDA) + Маринад Грузинский
- **Bundle:** Натуральные красители (карамель, бета-каротин), V Smart Plus Mg
- **Почему:** Натуральный дым-альтернатива = тренд премиум. NDA защитит рецептуру. Натуральные красители вместо нитритов.

## Типичные боли 2026
- Куриное мясо впитывает запахи → дорогая упаковка
- Потери на разморозке 2-3% + варке 12-15% = неприбыльно
- X5/Магнит требуют Clean Label, натуральный дым дорого
- Импортные маринады: минимум 500 кг, образцы 2-4 недели
- Импорт из Бразилии/ЕС дешевле на 15-20%

## Возражения и ответы
- **«Дорого»** → «Рассол окупается: -2% потерь при разморозке и -5% при варке. На 100 т сырья = 5 т продукции = +250 тыс ₽ выручки»
- **«Работают с импортом»** → «Образцы за 2 дня. Если конкурент замолчал хоть раз на 3+ дня — окно для нас»
- **«Нужен Clean Label, но фосфатов нет аналога»** → «Бесфосфатный (каррагинан + цитраты + растительные белки) даёт −2% к выходу vs фосфаты. Для ритейла это огромный плюс»
- **«Боятся изменения вкуса»** → «10-20 кг бесплатно. Технолог тестирует параллельно с текущим. Нравится — переходите»

## Конкуренты
- **Ван Хеес Рус (SPEKO)** — 200+ позиций для мяса и птицы. Слабость: 7-10 дней, дорого, старая немецкая школа
- **ЭКОС** — 700+ продуктов. Слабость: не доминирует на птице, нет скорости
- **Союзоптторг** — дистрибьютор. Слабость: только перепродажа, наценка, нет кастома

## Тренды 2026-2028
- **2026:** Clean Label и бесфосфатные технологии — требование X5/Магнит/Лента
- **2027:** Растительные заменители дыма вытесняют синтетические ароматизаторы
- **2028:** Функциональная птица (пребиотики, инулин, магний) для 40+ = мейнстрим

## Реальные клиенты (кейсы)
- **Рефтинская птицефабрика (Свердловская обл.)** — Clean Label, вход в X5
- **Птицефабрика «Заря» (Ленинградская обл.)** — бесфосфатный рассол, сохранение сочности
- **Федеральный производитель полуфабрикатов** — +4,5% выход (обезличено)

## Benchmark ориентиры
- Потери при разморозке: 2-3%, при варке 25-30% → с рассолом 12-15%
- Цена рассола: 85-130 ₽/кг (импорт), Verde 65-85 ₽/кг
- Скорость поставки конкурентов 10-14 дней vs Verde 5 дней
- Доля Clean Label: 10% (2024) → 35% (2026) → 60% (2028)
