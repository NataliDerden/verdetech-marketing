## Карта «задача → инструмент → артефакт»

### Блок 1. Графический контент (картинки, иллюстрации)

| Задача | Основной инструмент | Альтернатива | Вход / Выход |
|---|---|---|---|
| Продуктовые карточки (статик) | **Figma** + готовые шаблоны | Canva Pro | ТЗ + фото → PNG/JPG 1080×1350 |
| Инфографика | **Figma** + **Claude** (текст/данные) | Napkin AI, Piktochart | Данные → SVG/PNG |
| Иллюстрации и схемы | **Ideogram 3** (для текста на картинке), **Midjourney v7** | Recraft V3, Adobe Firefly | Prompt → PNG 2048+ |
| Обработка фото: фон | **Photoroom** | Remove.bg, Canva BG remover | JPG → PNG прозрачный |
| Апскейл / ретушь | **Topaz Photo AI** | Magnific, Gigapixel | Малый JPG → 4K |
| Цветокоррекция батчем | **Photoshop** (Neural Filters) + экшены | Lightroom AI | Папка → папка |
| Сведение композиции | **Figma** ИЛИ Photoshop Generative Fill | — | Элементы → макет |
| Слайды презентаций | **Gamma** (с импортом VERDE шаблона) | Beautiful.ai, Tome | Markdown/тезисы → .pptx/.pdf |

**Мастер-промпт для генераций** (Midjourney / Flux / Ideogram):
```
[subject: e.g., "close-up kale leaf"], clean minimal studio photography,
solid flat background color [mustard yellow #F2B42A | fresh green #0FBF00 | off-white #FCFCFC],
soft even lighting, no shadows, no texture, no grain, high detail,
B2B food ingredient brand aesthetic, Verde Tech style,
--ar 4:5 --style raw --stylize 50 --v 7
```
NEGATIVE: людей, рук, текста, водяных знаков, брендов, коллажа, сложных фонов.

### Блок 2. Видео и motion

| Задача | Основной | Альтернатива | Комментарий |
|---|---|---|---|
| Короткие вертикальные ролики (15-60 сек) | **CapCut** + AI-субтитры | VEED, Descript | Монтаж с AI-обрезкой пауз |
| Генерация видео с нуля | **Runway Gen-4** | Sora 2, Kling, Veo 3 | 5-10 сек клипы |
| Анимированные инфографики | **After Effects** + **Rive** | Canva Motion | Схемы, данные в движении |
| Моушн-плашки, титры, выноски | **After Effects** + шаблоны | Canva, CapCut | Единый стиль Onest + зелёный |
| Видеошаблоны (заставки, финалки) | **After Effects** (мастер-файл) | Motionleap | Создать 1 раз, команда использует |
| Субтитры + перевод | **Descript** | Submagic, CapCut | Автосубтитры Onest Medium |
| Адаптация под платформы | **CapCut** / **Opus Clip** | Gling | 1 ролик → 3 соотношения (9:16 / 1:1 / 16:9) |
| Аудио: озвучка, музыка | **ElevenLabs** (рус), **Suno** | HeyGen avatar | Для закадровых реплик |

**Шаблоны (сделать 1 раз):**
- Верт-ролик 9:16: заставка 2 сек с знаком VERDE → контент → финалка с вордмарком + слоганом + QR на сайт
- Плашка спикера: белая полоса внизу, имя капсом Onest Bold, должность Regular, знак справа
- Переходы: только wipe / cut. Никаких glitch, spin, zoom-blur.

|---|---|---|---|

**Мастер-шаблоны (создать и держать):**
- Каталог A4 — обложка + 4 типа разворотов (категория, продукт, технология, контакты)
- Листовка A5 — одна сторона продукт, вторая — ROI/факты
- Брошюра EuroFlyer (DL) — 3 сгиба, продуктовая линия

## Инструменты по категориям (быстрая сводка)

### Must-have (базовый набор на команду)
- **Figma** — основной рабочий стол, дизайн-система, все макеты
- **Claude** (API / Code / Web) — тексты, проверка, автоматизация
- **Midjourney** (Pro) — продуктовые фото, сцены, иллюстрации
- **Gamma** — быстрые презентации и КП
- **CapCut** (Pro) — все видео
- **Photoroom** — фон на продуктовых фото
- **Onest** — установлен у всех (бесплатно, SIL OFL)

### Nice-to-have (под задачи)
- **InDesign** — когда нужна серьёзная печать
- **After Effects** — моушн с брендом
- **Runway / Sora 2** — AI-видео
- **Ideogram** — когда нужен текст на картинке
- **v0.dev** — быстрые лендинги
- **Topaz Photo AI** — апскейл архивных фото

### Когда не нужны
- Canva — только если нужно, чтобы нетехнический сотрудник мог редактировать
- Stable Diffusion локально — если в команде нет ML-инженера, не стоит
- Adobe Creative Cloud полный — дорого, хватает InDesign + Photoshop
